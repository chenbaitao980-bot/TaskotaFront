# Spec: plan-card-scrollable-timeline

## 概述
AI 计划卡片内时间线模块和计划表模块在内容过多时支持滚动/分页，避免内容被遮挡。

## UI 规格

### 时间线模块 (#1)

**入口**：在 `_buildPlanCard` 中 `_buildPlanFlow` 渲染区域

**行为**：
- 固定最大高度 220px，超出可垂直滚动
- 时间线卡片以 `Wrap` 布局排列，同一行内相邻卡片保留箭头指示
- 换行后新行从头开始，箭头不跨行
- 右侧显示微弱的 Scrollbar（`ThemeData.useMaterial3` 默认 scrollbar）
- 行数 ≤ 3 时不需要滚动，自然展示

### 计划表模块 (#2)

**入口**：在 `_buildPlanCard` 中 `_buildPlanTable` 渲染区域

**行为**：
- 每页默认显示 5 行（表头不计入）
- 底部居中分页控件：`← 1/3 →`
- 分页状态按消息独立存储

## 验收标准
1. 时间线高度固定 220px，内容超出可垂直滚动
2. 计划表 0-5 行不分页，6+ 行分页
3. 分页控件可正常切换
4. 切换前后状态互不影响
5. 单行计划（≤5行）不显示分页控件
6. 回归测试记录

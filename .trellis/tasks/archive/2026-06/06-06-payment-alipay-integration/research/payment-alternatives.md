## 方案详细说明

### 1. 虎皮椒支付（xunhupay.com）— 推荐（无执照首选）

**定位**：专为个人开发者设计的收款聚合平台，对接支付宝/微信个人收款码

- **资质**：只需支付宝/微信账号实名认证，无需营业执照
- **原理**：通过监控支付宝/微信到账通知模拟异步回调（App轮询 or 云端监控）
- **手续费**：约 0.8%~1.2%
- **提现**：实时，无门槛
- **缺点**：
  - 依赖手机 App 运行，稳定性不如正规渠道
  - 属于"灰色"方案，有被封号风险
  - 金额不可自定义（需买家精确填写金额）
- **接入方式**：REST API + 回调，文档完善，与现有代码改动量中等

### 2. 支付宝当面付（正规）— 推荐（有执照后首选）

**这是项目代码已经实现的方案！**

- 代码完全写好：`supabase/functions/_shared/alipay.ts` 等
- 只需配置 4 个环境变量即可上线
- 需要：个体工商户营业执照 + 支付宝商家账号
- **建议路径**：先办个体户执照（几天），再直接用现有代码上线

### 3. 爱发电（afdian.net）— 最快启动

- **资质**：无需任何执照，注册即用
- **适合**：订阅制产品，用户在爱发电页面付款
- **缺点**：手续费高达 8%；无法嵌入 App 内（需跳转外链）；品牌感弱
- **适合场景**：MVP 早期测试，用户量极少时临时用

### 4. Stripe — 不推荐（国内场景）

- 国内用户需要有 Visa/MasterCard 才能支付，极大影响转化
- 不适合面向中国大陆用户的 App

## 推荐路径（按情况）

### 情况 A：想尽快合规 + 零手续费（推荐）

```
1. 线上办理个体工商户执照（1-5 天，免费或少量刻章费）
2. 用营业执照注册支付宝商家账号
3. 申请当面付能力（1-3 工作日审核）
4. 配置 Supabase 环境变量（ALIPAY_APP_ID 等 4 个）
5. 上线 ✅
```

**总成本**：¥0~200（执照费用） + 0% 手续费  
**时间**：约 1 周

### 情况 B：想本周就能收款（无执照临时方案）

```
1. 注册虎皮椒支付账号
2. 改造 create-order 和 order-status 函数对接虎皮椒 API
3. 临时上线收款
4. 后续办好执照后迁移回支付宝当面付
```

**总成本**：~1% 手续费  
**时间**：2-3 天开发

### 情况 C：极简 MVP 验证（不在意体验）

```
1. 注册爱发电，创建订阅套餐
2. App 内"开通VIP"按钮跳转爱发电链接
3. 用户付款后手动或 Webhook 更新订阅状态
```

**总成本**：8% 手续费  
**时间**：1 天

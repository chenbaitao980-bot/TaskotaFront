# 回归测试：日历任务完成态与周视图缩放

## 测试范围
- 日历周视图任务/子任务直接勾选完成
- 已完成日程和任务置灰显示
- 周视图 Ctrl+滚轮缩放时间轴

## 测试步骤

### 1. 任务完成态
1. 打开日历周视图
2. 点击任务块上的复选框
3. 验证任务立即置灰且状态变为已完成
4. 再次点击复选框，验证可取消完成

### 2. 多日任务完成态
1. 创建一个跨越多日的任务
2. 勾选该任务
3. 验证多日任务块全部置灰

### 3. 子任务完成态
1. 打开包含子任务的任务详情
2. 勾选子任务
3. 验证子任务置灰且父任务进度更新

### 4. 周视图缩放
1. 按住 Ctrl 键
2. 向前滚动鼠标滚轮
3. 验证时间轴缩小（显示更多时间段）
4. 向后滚动鼠标滚轮
5. 验证时间轴放大（显示更少时间段）

## 预期结果
- 所有完成态任务/日程显示为灰色弱化
- 缩放操作流畅，滚动位置保持语义正确
- flutter analyze 无新增警告

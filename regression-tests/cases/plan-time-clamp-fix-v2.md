# 回归用例：plan-time-clamp-fix-v2

## 背景
时间约束夹紧修复 v2：AI 生成日期归一化为本周连续日期。

## 覆盖范围
- AI 输出的显式日期（如 06-02）被夹紧到本周范围内
- 超出本周容量的计划行错峰分布（不同小时）
- 日期归一化为连续（无跳跃无重复）

## 手工验证步骤
1. 在 AI 助手中输入"只在早上踢球并且只在这周踢球"
2. 确认所有计划行的日期 ≤ 本周日（5月31日）
3. 确认日期连续无跳跃（5-25, 5-26, 5-27, ..., 5-31）
4. 确认时间线可横向滑动展示全部节点

## 通过标准
- 静态检查通过：`flutter analyze --no-fatal-infos`
- 用户验证：日期 ≤ 本周日、连续无跳跃、时间线可横向滑动

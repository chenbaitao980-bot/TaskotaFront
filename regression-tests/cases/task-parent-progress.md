# 回归测试：任务父任务展示与进度计算

## 测试范围
- 子任务详情页父任务展示与跳转
- 父任务进度按后代任务完成比例计算
- 多级嵌套任务进度计算

## 测试步骤

### 1. 父任务展示与跳转
1. 创建一个父任务
2. 在该父任务下创建一个子任务
3. 进入子任务详情页
4. 验证页面上方显示"父任务"区域，包含父任务标题
5. 点击父任务标题，验证跳转到父任务详情页

### 2. 父任务进度计算
1. 创建一个父任务，添加5个子任务
2. 将其中1个子任务标记为已完成
3. 进入父任务详情页
4. 验证进度显示为 20%

### 3. 多级嵌套进度计算
1. 创建父任务 A
2. 在 A 下创建子任务 B
3. 在 B 下创建子任务 C 和 D
4. 将 C 标记为已完成
5. 进入 A 的详情页
6. 验证进度计算基于所有后代（B、C、D），C 完成则进度为 33%

## 预期结果
- 子任务详情页正确显示父任务名称且可点击跳转
- 父任务详情页显示正确进度百分比
- 多级嵌套任务进度计算正确
- flutter analyze 无新增警告

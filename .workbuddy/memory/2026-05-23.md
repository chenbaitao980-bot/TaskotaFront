# 2026-05-23

## Windows 桌面应用构建（windows-desktop-build change）

- 执行 `flutter build windows --release`，成功生成桌面产物
- 产物路径：`build/windows/x64/runner/Release/smart_assistant.exe`
- 打包为 `smart_assistant_windows.zip`（13 MB，绿色版可直接分发）

### 构建注意事项（踩坑）
- **问题**：`flutter_local_notifications_windows` 插件需要 ATL 头文件（atlbase.h），VS BuildTools 未安装 ATL 组件导致编译失败
- **解决**：该插件在 lib/ 代码中未实际使用，注释掉 pubspec.yaml 中 flutter_local_notifications 和 timezone 依赖
- **环境变量**：需要在 PowerShell 中手动设置 `PROGRAMFILES(X86)` 环境变量，git bash 环境中此变量缺失
- Flutter 版本：3.41.9，VS BuildTools 2022（14.44.35207）

### 文件变更
- `pubspec.yaml`：注释 flutter_local_notifications 和 timezone 依赖
- OpenSpec change：`openspec/changes/windows-desktop-build/`（四件套完整）
- 回归测试：`regression-tests/cases/windows-desktop-build.md`、`runs/windows-desktop-build-latest.md`

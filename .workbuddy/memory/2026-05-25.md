## 话题漂移检测（已实施）

### 问题
用户从 ERP 开发讨论突然切换到"这周去踢球"，AI 仍然输出 ERP 计划表，不会自动切换话题。

### 三层修复
**Layer 1 - 客户端关键词漂移检测** (`ai_chat_page.dart`):
- `_detectTopicDrift()`：提取当前关键词，与最近 3 条用户消息逐条比对，零重叠→漂移
- `_extractKeywords()`：中文停用词+上下文时间词过滤
- `_isResponseText()`：回应类消息（没问题、继续等）不触发漂移

**Layer 2 - 历史批量切割** (`_callAI()`):
- 漂移→ `history = []`，传递 `topicSwitchDetected: true` 给 AIService

**Layer 3 - System Prompt 硬性规则** (`ai_service.dart`):
- 从 1 句话升级为检测标准、切换行为、禁止列表
- `topicSwitchDetected` 时注入头部警告

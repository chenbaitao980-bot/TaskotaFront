## calendar-filter-and-sync-fix — 日历筛选/同步/拖拽优化

修复了日历页面的 4 个问题：

### Bug 修复
1. **项目筛选无法切回"全部项目"**：`PopupMenuButton` 的 `onSelected` 不触发 `null` 值，改用 sentinel `'__all__'` 字符串代替 null
2. **日历拖拽后切换任务页面时间不同步**：日历修改任务后只更新本地状态未通知 `TaskNewBloc`，导致 TaskPage 数据未刷新。新增 `_notifyBloc()` 方法在每次修改后 `add(LoadTasks())`；home_page 切换 tab 时也触发刷新

### 优化
3. **时间线拖拽热区太小**：将 resize 从 `Draggable` 重构为 `GestureDetector` 方案（`_ResizableTaskBlock` 组件），热区从 12→20px，着色区可实时跟随拖拽变化
4. **跨日任务不可拖拽**：新增 `_EditableMultiDayBar` 组件，支持水平拖拽调整开始/结束日期、整体拖动移动

修改文件：`lib/presentation/pages/calendar/calendar_page.dart`、`lib/presentation/pages/home/home_page.dart`
